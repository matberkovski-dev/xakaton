# 🏗️ Инженерный Хакатон

Веб-система управления квестом: магазин материалов, задания, NFC-оплата, таймер игры, лидерборд.

## 🚀 Деплой на Render (бесплатно)

1. Форкните этот репозиторий на GitHub
2. Зайдите на [render.com](https://render.com) → New → Web Service
3. Подключите репозиторий
4. Настройки заполнятся автоматически из `render.yaml`
5. Нажмите **Deploy**

Через ~2 минуты сайт будет доступен по адресу `https://engineering-hakaton.onrender.com`

## 💻 Локальный запуск

```bash
pip install -r requirements.txt
python app.py
```

Откройте: http://localhost:5000

## ⚠️ Важно про базу данных

На Render бесплатном плане файлы в `/tmp` сбрасываются при каждом перезапуске сервиса (~15 мин неактивности). Данные хакатона сохраняются пока сервер работает. Для хакатона в реальном времени — этого достаточно.

Для постоянного хранения данных подключите PostgreSQL (платный план Render) или используйте локальный запуск.
